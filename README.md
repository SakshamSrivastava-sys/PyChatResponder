WhatsApp Auto-Reply AI

Description:
This project is a WhatsApp auto-reply system that automatically reads incoming messages, generates context-aware replies using AI, and sends them back without manual intervention. It helps streamline messaging, saves time, and demonstrates practical integration of Python automation with AI.

Features:

1. Automatically detects and reads incoming WhatsApp messages.

2. Generates intelligent responses using an AI API.

3. Sends replies automatically to the sender.

4. Configurable and secure API key management 

Tech Stack:

1. Programming Languages: Python

2. Libraries & Tools: PyAutoGUI, pyperclip, AI API (like OpenAI/Groq)

Other: Git, GitHub
